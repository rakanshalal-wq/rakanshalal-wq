<h2 align="right">مشاريعي الحالية</h2>

<table>
<tr>
<td width="50%">

<h3 align="right">رتّل — Rattl</h3>

منصة لتحفيظ القرآن بالذكاء الاصطناعي، تساعد الطالب وولي الأمر والشيخ على تجربة تسميع أوضح وأكثر تفاعلية.

<b>أعمل فيها على:</b>

- تجربة الطالب
- الذكاء الاصطناعي الصوتي
- جلسات التسميع
- لوحة الإدارة
- الربط والنشر السحابي

</td>
<td width="50%">

<h3 align="right">متين — Matin</h3>

منصة إدارة تعليم متكاملة للمؤسسات التعليمية، تجمع العمليات الأكاديمية والإدارية والمالية في لوحة تحكم واحدة.

<b>أعمل فيها على:</b>

- إدارة المدارس والجهات التعليمية
- الحضور والدرجات والتقارير
- الرسوم والمدفوعات
- الأدوار والصلاحيات
- النقل المدرسي والتكاملات
